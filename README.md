## 📚 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de desenvolvimento de APIs RESTful utilizando o ecossistema Spring Boot.

### Conteúdos abordados:

- Protocolo HTTP
- Verbos HTTP (`GET`, `POST`, `PUT`, `DELETE`)
- CRUD de APIs
- JSON e XML
- Spring Boot
- Maven
- JPA / Hibernate
- PostgreSQL
- Injeção de Dependência
- Tratamento de erros
- Validações

---

## 🔗 Exemplos de Endpoints

```http
GET    /alunos
POST   /alunos
PUT    /alunos/{id}
DELETE /alunos/{id}
```

---

## ⚙️ Como Executar o Projeto

```bash
# Clonar repositório
git clone <url-do-repositorio>

# Entrar na pasta
cd nome-do-projeto

# Instalar dependências
mvn clean install

# Executar aplicação
mvn spring-boot:run
```
